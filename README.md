# MCM - Sistema de Gestión de Cartera y Recaudos (MVP Fase 1)

Proyecto PHP + MySQL con login, roles, cargas de cartera, consulta, gestión, reportes y trazabilidad.

## Estructura
- `public/`: puntos de entrada web y assets.
- `app/`: configuración, middlewares, servicios y vistas.
- `sql/schema.sql`: DDL base + usuario admin inicial.
- `docs/`: manual técnico y de usuario.
